# MISSESSIN — Assistant Juridique

Assistant juridique en droit pénal et numérique béninois.

## Codes intégrés
- 📗 Code Pénal du Bénin (Loi N° 2018-16) — 988 articles
- 📘 Code de Procédure Pénale — 624 articles  
- 💻 Code du Numérique (Loi N° 2017-20) — 645 articles
- 🌐 Recherche web temps réel (jurisprudences)
- 📎 Analyse de documents PDF

## Déploiement sur Vercel

### Étape 1 — Créer un compte GitHub
1. Aller sur github.com
2. Cliquer "Sign up"
3. Créer votre compte avec votre email

### Étape 2 — Créer un dépôt GitHub
1. Cliquer le "+" en haut à droite → "New repository"
2. Nommer le dépôt : `missessin`
3. Laisser "Public" coché
4. Cliquer "Create repository"

### Étape 3 — Uploader les fichiers
1. Dans votre nouveau dépôt, cliquer "uploading an existing file"
2. Glisser tous les fichiers du dossier missessin-app
3. Cliquer "Commit changes"

### Étape 4 — Déployer sur Vercel
1. Aller sur vercel.com
2. Cliquer "Sign up" → choisir "Continue with GitHub"
3. Autoriser Vercel à accéder à GitHub
4. Cliquer "Add New Project"
5. Sélectionner le dépôt `missessin`
6. Laisser tous les paramètres par défaut
7. Cliquer "Deploy"

### Étape 5 — Votre URL
Vercel vous donnera une URL du type :
`https://missessin.vercel.app`

Partagez cette URL avec vos utilisateurs !
